@사용법
1. 크롤러를 실행합니다.
2. 자동으로 크롬에서 카테고리를 찾고, 작업이 끝나면 GUI가 켜집니다. (켜질 때까지 대기)
3. 좌측에 네이버 쇼핑의 큰 카테고리들이 버튼으로 써져있습니다. 원하는 카테고리의 버튼을 누릅니다.
4. 세부 카테고리들의 체크박스가 나오고, 크롤링할 체크박스만 체크합니다. (전체 체크를 누르면 모든 체크박스가 선택/해제 됩니다)
5. 우측에 카테고리들에서 검색할 페이지 수를 입력합니다. (최대 10)
6. 우측에 저장할 파일 이름을 입력합니다.
7. 시작 버튼을 누르고 크롤링이 종료될 때까지 대기합니다.

@주의
- 네이버에서 크롤링을 단시간에 많이 할 경우 페이지 폴트를 발생시켜서 중간 중간에 데이터가 비는 경우가 있음을 확인했습니다.
  최대한 발생하지 않도록 중간 중간에 쉬는 시간을 넣어두었지만 혹시라도 비어 있는 정보가 있다면 해당 카테고리만 따로
  한 번 더 크롤링 하는 것을 추천드립니다. (해결법을 찾으면 수정하겠습니다)
- 엑셀 파일을 보면 중간 중간에 가격비교 정보들이 비어 있는 곳이 있을텐데, 링크를 넣어 두었습니다.
  페이지 폴트가 발생한 것이 아니라, 가격 비교로 들어갔지만 바로 다른 페이지로 이동하거나, 가격 비교 형식이 아닌 독특한
  형식의 링크일 것입니다.
- [면세점] 카테고리에는 가격 비교가 없습니다. 하지만 체크박스는 넣어두었습니다. 크롤링할 때 넣지 않는 것을 추천드립니다.
  빈 정보만 엑셀 시트를 차지할 뿐입니다.
- chromedriver.exe 버전은 80입니다. 폴더에 들어있고, 혹시 오류가 난다면 본인 컴퓨터 크롬 버전에 맞는 드라이버를 설치해서
  교체해주세요. (링크 : https://chromedriver.chromium.org/)


@추신
- UI에 대한 피드백이 있으시면 원하시는대로 수정할 수 있도록 해보겠습니다. (시안을 그려서 주시면 좋을듯 합니다)
- 이전에 했던 금융 크롤러와는 다르게 쇼핑에서는 네이버에서 걸어둔 제약이 많아 보입니다. 그래서 생각 외의 예외가 많을듯 예상되는데,
  저도 지속적으로 테스트하면서 잡아보겠습니다. 혹시 사용하는데 문제가 되는 오류가 있다면 작은 것이라도 말해주시면
  수정하도록 하겠습니다.
